# 短剧创意生成器 — Short Drama Concept Creator

> 将影视策划经验与短剧市场洞察编码为 AI 可执行的结构化创意方法论引擎。
> 单文件零依赖，浏览器打开即用，兼容任意大模型。

## 这是什么

一个**创意激发工具**——生成经过质检的高质量创意概念，帮创作者从多个方向中找到最值得继续的那个，然后继续打磨。

你选参数（受众、题材、情绪）→ 工具把 7 大规则体系打包成结构化 Prompt → 交给大模型生成 3 个经过质检的创意概念 → 你做判断（选哪个、放弃哪个）→ 反馈沉淀到你的创作者画像 → 下次更精准。

**解决的核心问题：** 现有 AI 创作工具面向批量生产（日产几十部的 MCN），输出的是"一段剧本"。这个工具相反——垂直聚焦女频、精品导向，输出的是"经过质检的创意概念"。AI 负责生成和质量底线，人负责判断和审美上限。

## 核心理念

好的创意不是凭空生成的，也不是简单复制市场爆款。好的创意来自**市场规律**和**创作者个人特质**的交汇点。

这个工具做两件事：
1. **提炼市场规律**——基于真实爆款数据，找到背后的结构性规律，编码为 AI 可执行的规则链
2. **适配创作者**——通过反馈沉淀，逐渐了解你的审美偏好，让每次激发更贴合你，最终找到只属于你的创作方向

## 内置方法论

基于 2025H2-2026Q1 短剧市场爆款观察提炼的 7 大规则体系：

| 规则 | 核心内容 |
|------|----------|
| 规则零 | 从幻想出发，不从问题出发（三种幻想：能力/身份/行动） |
| 喜剧公式 | 奇幻触发器 × 极端身份错位 × 家庭关系 × 喜剧 × 治愈（至少 3/5 通过） |
| 大女主原则 | 观众追的是女主不是 CP，每个转折是她的决定 |
| 标题 5 铁律 | 有情绪承诺 / 有反差 / 有动词 / 4 字或长句 / 迭代高频词 |
| 概念 3 铁律 | 一句话 = 情绪过山车 / 核心是她做了什么 / 兴奋不是沉重 |
| 女主三关 | 配得感 / 快意恩仇 / 主体性 |
| 常见错误 | 避免太实的职场戏 / 电影文艺片感 / 自媒体 vlog 感 |

完整方法论详见 → [methodology.md](methodology.md)

## 使用方式

1. 下载 `index.html`，用浏览器打开
2. 选择受众、题材、情绪标签
3. 点击「生成创意 Prompt」，复制生成的 Prompt
4. 粘贴到任意大模型（MiMo、ChatGPT、Claude、混元等）
5. 看到结果后回来写反馈
6. 反馈自动沉淀到你的「创作者画像」
7. 下次生成时画像自动附加，循环往复

```
选参数 → 生成 Prompt → 大模型跑出结果 → 找到最值得继续的方向 → 写反馈 → 画像更新 → 继续打磨
```
