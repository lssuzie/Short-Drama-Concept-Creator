# 交接：故事创意炼金坊 — 精致化与通用化升级

本项目已完成从“精品女频短剧”向**“故事/短剧/漫剧/电影创意炼金术 (Story Alchemy)”**的架构与美学升级，并在细节交互、i18n 多语言和云端同步机制上实现了全面优化。

---

## 已完成

### 1. 故事创意通用化与题材扩展
* **方法论大女主内核平移**：原汁原味地保留了“大女主原则”与“女主三关”的硬核标准。在适用男频或通用故事时，系统提示词会在下游自动平滑映射为高品质的“大男主/他”，规避无脑系统或打脸套路。
* **“热血逆袭”男频预设闭环**：
  * 主界面快速示例新增了 `🔥 热血逆袭` 按钮。
  * 补齐了预设中包含的核心情绪标签：`热血燃向`、`反转逆袭` 以及根据用户反馈打磨替换的 `逆风翻盘`（取代较俗套的“莫欺少年穷”）。
  * 题材下拉框中新增了 `热血逆袭` 与 `科幻脑洞`。

### 2. 全局无死角双语（i18n）切换
* **多语言词典匹配修复**：修正了 slogan 句式变动和标点符号导致的中英混杂 Bug。
* **Input & Textarea Placeholder 动态翻译**：扩展了 `snapshotPage()` 与 `applyLang()` 的树遍历范围，增加了对所有 `placeholder` 属性的拦截翻译。
* **默认值自动平移**：在语言切换时，若“目标受众”依然是默认值，会自动在 `"女频 25-35岁"` 与 `"Female 25-35"` 之间进行智能语言平移，不破坏用户已修改的内容。

### 3. 透光毛玻璃与高端交互视觉 (Premium UI/UX)
* **Glassmorphism 视觉升级**：
  对设置面板 (`.settings-panel`)、认证弹窗 (`.modal-content`)、数据看板、质检卡片及反馈卡片，引入了 `backdrop-filter: blur(12px)` 磨砂质感和半透明背景。
* **微交互 Hover & Glow 发光**：
  所有核心按钮与复选框标签在 Hover 时增加 `1-2px` 的物理平滑浮动与背影阴影；Checkbox 选中时有微弱的外发光阴影 (Glow)。
* **长文阅读体验优化**：
  将 Prompt 激发框 (`#obox`) 的字号调整为 14px，背景设为低对比度黑灰，文字设为高对比度的白骨色 (`var(--tp)`)，行高调宽，大幅减轻阅读疲劳。

### 4. Supabase 云端同步与数据闭环 (已解决遗留问题)
* **云端删除同步 (deleteHist)**：
  点击本地卡片的“✕”删除时，若用户已登录，系统不仅移除本地缓存，还会根据时间戳与用户 ID **自动向 Supabase 发送 delete 请求**，实现云端历史记录同步清理。
* **清空历史二次确认 (clearHist)**：
  对本地创意库「清空」按钮增加了 `t('确定清空所有本地历史？')` 二次确认弹窗，防止用户误触清空所有草稿。

## 交接要点
* 本项目为**单文件零依赖**网页应用，编译后仅生成单个根目录 `index.html`，易于携带和本地离线运行（本地优先，登录后自动通过网络同步云端）。
* 所有关于视觉、交互与方法论的更改必须首先在 `src/` 目录下完成，然后运行 `node build.mjs` 进行打包，不要直接修改根目录的 `index.html`。
